# **Privacy Policy**
Last updated: 2026-03-14

This privacy policy applies to the mobile application **DogDex**, developed by Toby van der Peet.

### 1. Information We Collect

DogDex does not collect personal information such as name, email address, or account details.

When the app is used, it sends requests to a backend API operated by the developer.
The API may log limited technical information including:

* Request time
* Requested API endpoint

This information is used only for debugging, analytics, and service improvement.
No personally identifiable information is intentionally collected or stored.

### 2. How Information Is Used

Logged request data is used to:

* Monitor API performance
* Diagnose errors
* Improve the functionality of the app

The data is not used to identify users.

### 3. Data Sharing

No personal data is sold, shared, or provided to third parties.
Request logs are stored only on the developer’s server.

### 4. Data Storage and Retention

Request log data may be stored temporarily for analytics and debugging purposes.
Logs may be deleted at any time and are not kept longer than necessary.

### 5. Security

Reasonable measures are taken to protect stored data from unauthorized access.
Since no personal information is collected, the risk to users is minimal.

### 6. Children’s Privacy

DogDex does not knowingly collect personal information from children.
If any such data is discovered, it will be deleted.
